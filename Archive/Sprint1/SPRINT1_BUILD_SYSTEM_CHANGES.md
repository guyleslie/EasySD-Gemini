# Sprint 1 - Build System Changes

## Összefoglaló

A build rendszer kibővítése három különálló build target támogatásával, amelyek egyértelműen jelzik a fejlesztési célt és automatikusan konfigurálják az Arduino DEBUG módot.

---

## Változtatások Áttekintése

### 1. **Új Build Targetok**

| Target | C64 DEBUG | Arduino DEBUG | Cél |
|--------|-----------|---------------|-----|
| **release** | ❌ OFF | ❌ OFF | Production build |
| **debug-vice** | ✅ ON | ❌ OFF | C64 menü fejlesztés VICE debuggerrel |
| **debug-arduino** | ✅ ON | ✅ ON | Arduino firmware tesztelés Serial Monitorral |

**Használat:**
```bash
# Production build (mindkét DEBUG OFF)
python build.py release

# C64 VICE debug (mock data, Arduino Serial OFF)
python build.py debug-vice

# Arduino Serial debug (Serial.println() enabled, teszt parancsok)
python build.py debug-arduino

# Core + plugins külön (opcionális)
python build.py core
python build.py plugins
```

## Működés Részletesen

### Build Folyamat (debug-arduino target):

1. **build.py futtatás:**
   ```bash
   python build.py debug-arduino
   ```

2. **Flagek beállítása:**
   - `debug = 1` (C64 DEBUG: VICE mock data)
   - `arduino_debug = 1` (Arduino Serial DEBUG)
   - `build_arduino = True`

3. **C64 assembly fordítás:**
   - `64tass -D DEBUG=1 ...` → irqhack64-debug.prg
   - Plugins fordítása DEBUG=1-gyel

4. **Arduino artifacts generálás:**
   - FlashLib.h létrehozása (C64 menu data)
   - **BuildConfig.h generálás:**
     ```cpp
     #define DEBUG
     ```
   - Másolás `Arduino/IRQHack64/` mappába

5. **Arduino IDE upload:**
   - Nyisd meg: `Arduino/IRQHack64/IRQHack64.ino`
   - IrqHack64.h include-olja a BuildConfig.h-t
   - DEBUG definiálva van → Serial.println() működik
   - Upload (Ctrl+U)
   - Serial Monitor @ 57600 baud

## Hibakeresés

### Probléma: "Sketch too big" hiba Arduino IDE-ben

**Ok:** DEBUG mód bekapcsolva (175+ Serial.print hívás = ~10 KB overhead)

**Megoldás:**
```bash
# Használd debug-vice vagy release targetet
python build.py debug-vice   # vagy release

# Ellenőrzés:
cat Arduino/IRQHack64/BuildConfig.h
# Kimenet: "// DEBUG disabled ..."

# Arduino IDE: Upload újra
```

---

### Probléma: Serial Monitor nem mutat semmit

**Ellenőrzés:**
```bash
# 1. Ellenőrizd hogy debug-arduino build volt-e
cat Arduino/IRQHack64/BuildConfig.h
# Elvárás: #define DEBUG

# 2. Ha nincs, build újra:
python build.py debug-arduino

# 3. Arduino IDE: Upload újra
```

---

### Probléma: BuildConfig.h manually edited

**⚠️ NE szerkeszd kézzel a BuildConfig.h-t!**

Ha mégis módosítottad:
```bash
# Regeneráld a következő build-del
python build.py debug-arduino   # vagy debug-vice / release
```

A build.py **mindig felülírja** a BuildConfig.h-t!
