HARDVER SETUP:
--------------
Arduino Nano ──┬─ D10 (CS)   ─→ SD Module CS
               ├─ D11 (MOSI) ─→ SD Module MOSI  ⚠ NE CSERÉLD FEL!
               ├─ D12 (MISO) ─→ SD Module MISO  ⚠ NE CSERÉLD FEL!
               ├─ D13 (SCK)  ─→ SD Module SCK
               ├─ 5V         ─→ SD Module VCC
               └─ GND        ─→ SD Module GND

⚠ KRITIKUS: D11=MOSI, D12=MISO (NE FORDÍTVA!)

KÓD VALIDÁLÁS:
--------------
✓ IRQHack64.ino használja a Hardware SPI-t (#include <SPI.h>)
✓ MOSI/MISO/SCK automatikusan beállítva (hardver SPI controller)
✓ CS pin explicit definiálva: chipSelect = 10
✓ SPI inicializálás: sd.begin(chipSelect, SPI_HALF_SPEED)

PIN FUNKCIÓK (ATmega328P):
---------------------------
D11 (PINB3/MOSI): Arduino → SD adatküldés
D12 (PINB4/MISO): SD → Arduino adatfogadás
D13 (PINB5/SCK):  Clock jel szinkronizáció
D10 (SS/CS):      Chip Select (eszköz választás)

SD KÁRTYA KÖVETELMÉNYEK:
------------------------
- Formátum: FAT32 (32KB allocation unit)
- Méret: Max 32GB
- Típus: SD/microSD (nem SDHC exFAT)
- 8.3 fájlnevek (rövid nevek)

TESZTELÉSI PARANCSOK (Sprint 1):
