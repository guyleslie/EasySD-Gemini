#==============================================================================
# EasySDMenu.s - C64 Menu Program for EasySD Cartridge
# -----------------------------------------------------------------------------
# Purpose: Main menu program for the EasySD cartridge. Handles user navigation,
#          directory listing, and plugin launching on the Commodore 64. Uses
#          the cartridge API for all file and directory operations.
#
# Key Architecture (2026):
#   - Menu is loaded only on explicit SEL button press (never at cold boot)
#   - All directory and file operations are performed via cartridge protocol
#   - Plugin system: menu launches plugins based on file type (PRG, KOA, WAV, etc.)
#   - Robust navigation: supports paging, directory up/down, and error handling
#
# Lessons Learned (2026):
#   - Always document menu entry/exit and plugin invocation flow
#   - Never assume directory state; always refresh from firmware after changes
#   - All error and state transitions must be visible to the user
#   - Use symbolic names for all ZP and RAM variables (see CartZpMap.inc)
#
# See also: docs/ARCHITECTURE_OVERVIEW.md, CartLib.s, CartApi.cpp
#==============================================================================
