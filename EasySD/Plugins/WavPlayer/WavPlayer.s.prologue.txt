#==============================================================================
# WavPlayer.s - C64 WAV Audio Plugin for EasySD
# -----------------------------------------------------------------------------
# Purpose: Plugin for playing 4/8-bit WAV audio files on the C64 using the
#          EasySD cartridge. Supports multiple playback modes (SID, DigiMax,
#          MK3) and double-buffered streaming from SD card via Arduino.
#
# Key Architecture (2026):
#   - Uses IRQ-driven streaming for smooth audio playback
#   - Supports both IO2 streaming and NMI-based MK3 modes
#   - All buffer and ZP usage is documented and follows CartZpMap.inc
#   - Robust error handling: always returns to menu on failure
#
# Lessons Learned (2026):
#   - Always document buffer layout and ZP usage for plugins
#   - Never assume SD or audio hardware is present; always detect and fallback
#   - All plugin entry/exit and error paths must be clear and BASIC-safe
#   - Use macros/APIMacros for all protocol operations
#
# See also: docs/PLUGIN_QUICKSTART.md, CartLib.s, APIMacros.s
#==============================================================================
